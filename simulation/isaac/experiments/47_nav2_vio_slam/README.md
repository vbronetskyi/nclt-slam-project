# exp 47: Nav2 + VIO SLAM Localization

## результат

Nav2 + VIO SLAM closed-loop навігація на повному outbound маршруті (193м): ATE 0.150м, scale 1.003, 2604 frames, 0 lost. 33/39 WPs reached.
