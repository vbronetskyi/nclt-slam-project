# exp 01: NVIDIA Rivermark outdoor environment

спробував завантажити готову сцену NVIDIA Rivermark в Isaac Sim щоб
перевірити чи буде вона придатна для visual-inertial SLAM експериментів
з Husky A200

## результат

сцена відкрилася, але виявилась занадто щільною на GPU (RTX 3090
тягнув десь 8 fps з husky + сенсорами зверху).  ще там було забагато
дрібних деталей які не потрібні для нашого тесту, тому переключились на
власну procedural-сцену в наступних experiments
